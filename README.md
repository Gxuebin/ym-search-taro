# ym-search-taro
taro多端小程序

基于taro 2.0.0  taro-ui 2.2.4编写的适配多端小程序项目，仅用于交流学习

功能  

 创建taro开发页面模板  
  npm run tpl 页面模块名称  页面模块路径
   example:  npm run tpl taroPage ./src/pages/     将创建模板到./src/pages/路径下
